Instalar

Ubuntu

Puedes añadir el PPA del proyecto e instalarlo desde ahí 
	sudo add-apt-repository ppa:infjaf/dalle
	sudo apt-get update
	sudo apt-get install dalle

Descarga desde la página de GitHub

Puedes descargarte el programa ya compilado así como las fuentes de la página de descargas del proyecto en Github

En la página de descargas encontrarás:

  • dalle_VERSION_all.deb Paquete para instalar en distribuciones basadas en DEB (como Debian o Ubuntu). Para ubuntu es recomendable usar el PPA indicado arriba
  • dalle-VERSION.noarch.rpm. Paquete para instalar en distribuciones basadas en RPM (RedHat, Fedora, Mandrake ...)
  • dalle-VERSION.tar.bz2. Contiene las fuentes.
  • dalle-VERSION-windows.zip. Contiene una version especifica para windows y .NET
  • dalle-VERSION-bin.zip. Contiene los programas las y librerias compilados, así como la libreria ICSharpCode.SharZipLib.dll, necesaria si trabajas con MS.NET.

Descargar las fuentes de GitHub

Si quieres descargarte la última versión de github, solo tienes que clonar el repositorio.

    git clone https://github.com/albfernandez/dalle.git
